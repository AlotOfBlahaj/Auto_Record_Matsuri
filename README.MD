Auto_Record_Matsuri
====
此程序能够自动监控夏妹直播并下载当前的直播视频（其实你也可以修改其中的URL来监控其他VTB[臭DD]

    支持
    Youtube
    Mirrativ
    

需要
------
- 手动安装FFMPEG并设置PATH
- 首次启动时请先运行install
- 你可以运行run打开程序

支持
------
- 如需更改存储目录和代理地址，请编辑config.py
